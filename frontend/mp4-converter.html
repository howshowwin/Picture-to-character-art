WebM to MP4 轉換指南

由於瀏覽器限制，無法直接錄製 MP4 格式。以下是轉換方案：

為什麼需要轉換？

大多數瀏覽器的 MediaRecorder API 只支援 WebM 格式錄製。要獲得 MP4 格式，需要進行後續轉換。

轉換方案

方案 1：使用 FFmpeg（推薦）

FFmpeg 是最強大的視頻轉換工具。

# 基本轉換 ffmpeg -i character-art-1000x1000.webm -c:v libx264 -preset fast -crf 22 character-art-1000x1000.mp4 # 高品質轉換 ffmpeg -i character-art-1000x1000.webm -c:v libx264 -preset slow -crf 18 -pix_fmt yuv420p character-art-1000x1000.mp4 # 保持原始品質 ffmpeg -i character-art-1000x1000.webm -c:v copy character-art-1000x1000.mp4

下載 FFmpeg

方案 2：線上轉換工具

  • CloudConvert - 免費且快速
  • Convertio - 簡單易用
  • Online Converter - 無需註冊

方案 3：使用 VLC 媒體播放器

  1. 開啟 VLC
  2. 媒體 → 轉換/儲存
  3. 加入 WebM 檔案
  4. 選擇 "轉換"
  5. 選擇 MP4 格式
  6. 開始轉換

下載 VLC

方案 4：使用 HandBrake

專業的開源視頻轉換軟體，提供詳細的編碼選項。

快速 Python 轉換腳本

如果您有 Python 環境，可以使用以下腳本批量轉換：

import subprocess import os from pathlib import Path def convert_webm_to_mp4(input_file, output_file=None): """Convert WebM to MP4 using ffmpeg""" if output_file is None: output_file = Path(input_file).with_suffix('.mp4') cmd = [ 'ffmpeg', '-i', input_file, '-c:v', 'libx264', '-preset', 'fast', '-crf', '22', '-pix_fmt', 'yuv420p', output_file ] subprocess.run(cmd) print(f"Converted: {input_file} -> {output_file}") # 使用範例 convert_webm_to_mp4('character-art-1000x1000.webm')
