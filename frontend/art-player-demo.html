🎨 Character Art Player with GSAP

簡單範例 - 基本用法

// 基本用法
playCharacterArt({
    path: 'test-animation.json',
    width: 80,
    canvasClass: 'my-art-canvas'
});

進階範例 - 自訂選項

// 進階用法
const player = await playCharacterArt({
    path: 'animation.json',
    width: 100,
    canvasClass: 'custom-canvas',
    fps: 30,
    loop: true,
    color: '#00ff00',
    fontSize: 14,
    backgroundColor: '#000000',
    autoplay: true,
    onFrame: (current, total) => {
        console.log(`Frame ${current + 1}/${total}`);
    },
    onComplete: () => {
        console.log('Animation completed');
    }
});

API 方法說明

// 可用方法
player.play()      // 播放動畫
player.pause()     // 暫停動畫
player.resume()    // 繼續播放
player.stop()      // 停止並重置
player.seek(10)    // 跳到第 10 幀
player.setSpeed(2) // 設定播放速度
player.destroy()   // 銷毀播放器
